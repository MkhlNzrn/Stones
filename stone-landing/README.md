# Лендинг «Изделия из камня»

Сайт для студии по изготовлению изделий из камня (столешницы, камины, лестницы и т.д.).

## Запуск

1. Установить зависимости:
   ```bash
   cd stone-landing
   npm install
   ```

2. (Опционально) Настроить отправку заявок на почту:
   - Скопировать `.env.example` в `.env`
   - Для Gmail: включить двухэтапную аутентификацию и создать «Пароль приложения»
   - Указать в `.env`: `SMTP_USER` и `SMTP_PASS`

3. Запустить сервер:
   ```bash
   npm start
   ```

4. Открыть в браузере: http://localhost:3000

## Фотографии на сайте

- **`photos`** — примеры работ. Добавьте сюда файлы изображений (jpg, png, gif, webp), они появятся в блоке «Примеры наших работ».
- **`stones`** — фото типов камней. Внутри должны быть папки с именами: **кварц**, **кварцит**, **мрамор**, **оникс**, **травертин**, **гранит**, **арагонит**, **брекчия**. Положите в каждую папку хотя бы одно фото — оно будет показано в блоке «Материалы».
- **`items`** — фото изделий. Папки: **столешницы**, **столы**, **камины**, **лестницы**, **подоконники**, **полы**, **прочее**. В каждой — хотя бы одно изображение для блока «Что мы изготавливаем».
